# ChemKit

Chemkit is a Python package that provides equations and tools built to aid Chemists of all levels in their understanding of the chemical nature of the world. It aims to be useful for carrying out practical, real world calculations and modelling; a useful open source tool available to everyone who needs it.

## Features
- Solve basic Chemistry equations such as Avogadro's equation, gas laws and more efficiently.
- Provide a list of units and constants typically used in Chemistry.
- Provide algorithms that combine fundamental equations to streamline and accelerate chemical computations.

## How to install

ChemKit is not yet published on PyPI, but you can install it directly from the source.

### 1. Clone the repository
```bash
git clone https://github.com/Lonercode/chemkit.git
cd chemkit
```

### 2. Create and activate a virtual environment (recommended)

```bash
python -m venv venv
```
```bash
venv\Scripts\activate # For windows
```
```bash
source venv/bin/activate # macOS/Linux
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```
